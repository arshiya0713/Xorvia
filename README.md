# Xorvia — Attendance Management System

A web-based attendance management system built with Angular, PHP, and MySQL.

## Tech Stack

- **Frontend:** Angular 17 (Standalone Components)
- **Backend:** PHP
- **Database:** MySQL (via XAMPP)

## Features

- Role-based login (Admin / Employee)
- QR code based attendance marking (rotates every 30s)
- Real-time dashboard with attendance stats
- Employee management (Add / Edit / Delete)
- Monthly attendance reports with CSV export
- Auto marks absent for previous day if not checked in
- Password change functionality

## Prerequisites

- Node.js 18+
- Angular CLI (`npm install -g @angular/cli`)
- XAMPP (Apache + MySQL)
